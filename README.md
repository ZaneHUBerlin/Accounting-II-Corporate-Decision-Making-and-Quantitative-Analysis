# Accounting II – Assignment 2

## Topic
Correlation between the Price-to-Book Ratio and Return on Assets (ROA)

## Data
- Source: Compustat North America
- Fiscal year: 2023
- Sample: Public non-financial firms
- Note: Raw data are not uploaded due to licensing restrictions.

## Files
- `analysis.R`: R script for data cleaning, correlation analysis, and visualization
- `scatter_roa_pb.png`: Scatter plot of ROA and Price-to-Book
- `descriptive_statistics.csv`: Summary statistics
- `correlation_results.csv`: Pearson and Spearman correlations
